# EPILEPSUM
Más de 300.000 ecuatorianos al año, y cada uno de estos sucesos transcurre de diferente manera, surge una inseguridad al momento de sufrir una crisis fuera de casa sin saber de la manera correcta de como poder ayudar a las personas en esos momentos de crisis.

La solución planteada en este momento es la creación de una aplicación web que contiene  un horario de control de medicación para evitar estas crisis, dando una guía de primeros auxilios para cuando una  persona sufre de epilepsia fuera de casa, dando la ubicación exacta a un contacto de emergencia registrado.
